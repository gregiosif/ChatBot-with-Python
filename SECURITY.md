# 🔐 Security Policy

## 📦 Supported Versions

This project is currently in its initial release phase. Security updates and fixes will be provided for the latest version available in the main branch.

## 🛡 Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

- **How to report**: Open a GitHub issue and label it as `security`, or contact the maintainer via GitHub profile.
- **Response time**: You can expect a reply within 5–7 business days.
- **What happens next**:
  - The issue will be reviewed and verified.
  - If valid, a fix will be developed and released.
  - You may be credited (if desired) in the changelog or release notes.

Please avoid publicly disclosing vulnerabilities until they are resolved.
